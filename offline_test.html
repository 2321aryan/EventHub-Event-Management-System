In XAMPP Control Panel:

  1. Click "Start" next to Apache
  2. Click "Start" next to MySQL

C:\xampp\htdocs\eventhub\

The folder should contain files like: login.php, index.html, style.css

Step 4: Test the Application

Once Apache is running, try these URLs:

  • http://localhost/eventhub/
  • http://localhost:8080/eventhub/
